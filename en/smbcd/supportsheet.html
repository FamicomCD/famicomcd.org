Since SMBCD is a rom hack and not a computer game, emulator and hardware support is very important for a smooth and intended experience. Emulators that don't support MSU-1 at all will cause the MSU-1 Required screen to come up. If you're looking for SD Card support sheets for the FXPak Pro/SD2SNES, please check the SD2SNES Card List.

If you're using an emulator or hardware that isn't shown on the list below, feel free to submit it via the FamicomCD Discord, or by emailing at FamicomCD@gmail.com. This table doesn't work well on mobile in portrait mode; use landscape mode for a good experience. Also, please assume that all emulators mentioned are using the latest version, and that emulators not listed here are unsupported unless stated otherwise.
